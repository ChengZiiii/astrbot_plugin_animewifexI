# Phase 3 完整 QA 测试用例（v3.0.0-phase3-final）

测试环境：AstrBot 实例，群聊中发送命令

---

## 快速 QA 流程（管理员）

### 第 1 步：准备
```
老婆 重置本群
```
清空所有数据，开始干净测试。

### 第 2 步：测试抽老婆 + 稀有度
```
抽老婆
```
确认显示：老婆图片 + 稀有度（✨/🌟/⭐/·）

### 第 3 步：测试重置抽卡
```
老婆 重置抽卡
抽老婆
```
确认可以重新抽，且老婆不同（或相同但重新分配稀有度）。

### 第 4 步：测试经济系统
```
老婆 签到
老婆 任务
老婆 商城
老婆 购买 换老婆券
老婆 背包
```
确认：签到成功 → 任务显示进度 → 商城列表 → 购买扣币 → 背包显示道具

### 第 5 步：测试亲密度 + 求婚
```
老婆 测试亲密度 @自己 70
老婆 求婚 1
```
确认：亲密度设置成功 → 求婚成功（扣 100 币，永久锁定）

### 第 6 步：测试锁定/解锁
```
老婆 测试币 @自己 200
老婆 购买 锁定卡
老婆 锁 1
老婆 解锁 1
```
确认：锁定成功 → 解锁成功

### 第 7 步：测试 PK
```
老婆 测试抽卡 1
```
（让另一个群友也有老婆后）
```
老婆 PK @对方
```
确认：战报显示双方战力 + 胜者 + 奖励

### 第 8 步：测试图鉴 + 面板
```
老婆 图鉴
老婆 面板
```
确认：图鉴显示收集数/稀有度统计；面板显示持有老婆/统计/币

### 第 9 步：测试管理员命令
```
老婆 测试币 @自己 999
老婆 测试亲密度 @自己 99
老婆 测试抽卡 100
```
确认：余额/亲密度修改成功；模拟抽卡显示概率分布

---

## 管理员命令速查

| 命令 | 用途 |
|---|---|
| `老婆 重置本群` | 清空本群所有数据 |
| `老婆 重置抽卡 [@某人]` | 重置今日抽卡状态 |
| `老婆 测试抽卡 [次数]` | 模拟抽卡（不写入数据） |
| `老婆 测试亲密度 @某人 <数值>` | 设置亲密度 |
| `老婆 测试币 @某人 <数值>` | 设置老婆币余额 |

## 验收清单

- [ ] 抽老婆显示稀有度
- [ ] 重置抽卡后可重新抽
- [ ] 签到/任务/商城/购买/背包 正常
- [ ] 求婚成功（亲密度 ≥ 60 + 100 币）
- [ ] 锁定/解锁正常
- [ ] PK 正常（需两个有老婆的用户）
- [ ] 图鉴/面板正常
- [ ] 管理员测试命令正常

测试完成后告知结果。
